English Study Hub 사용법

1) 시작하기
- english-study-site 폴더 안의 index.html 파일을 브라우저로 열면 바로 사용 가능합니다.
- 별도 설치가 필요 없는 정적 웹페이지입니다.

2) 파일 구성
- index.html : 페이지 구조
- styles.css : 디자인
- script.js : 버튼 동작, 표현 변경, 단어 섞기, 퀴즈, 일기 임시저장

3) 수정하기 좋은 부분
- 오늘의 표현: script.js 의 expressions 배열 수정
- 단어 카드: index.html 의 vocab-card 블록 수정
- 주간 기록: index.html 의 log-card 수정
- 색상/분위기: styles.css 의 :root 변수 수정

4) 배포
- 도메인 연결 없이 GitHub Pages 기본 주소로만 올려도 됩니다.
- 또는 그냥 로컬 파일로만 사용해도 됩니다.

5) 추천 확장 기능
- MP3 듣기 버튼
- 단어장 CSV 불러오기
- 오답노트 저장
- 오늘의 문장 음성 녹음 링크 연결
