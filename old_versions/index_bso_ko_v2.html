에너지 시스템, AI, 클라우드 인프라를 함께 공부하며 실제로 작동하는 구조를 만드는 데 관심이 있습니다. 이곳은 학사·석사 과정에서 수행한 연구, 팀 프로젝트, 실험 기록을 정리하는 개인 홈페이지입니다.

태양광 무선충전 자율주행 실험, 보안 게이트웨이 구축, GitOps 기반 클라우드 인프라, 수소 혼소 화염 모니터링, 저류층 시뮬레이션과 프록시 모델링 같은 주제를 다룹니다.

에너지 시스템

오일샌드 열회수, 메탄하이드레이트, 수소 혼소처럼 에너지 생산과 운영 문제를 모델링과 실험 관점에서 다룹니다.

적용형 AI

OpenCV, CNN, SSD, YOLO 기반으로 영상 인식과 예측 모델을 만들고, 실제 장치나 운영 시나리오에 연결하는 데 관심이 있습니다.

클라우드 인프라

GitOps, 모니터링, 백업, 보안 게이트웨이처럼 시스템을 배포하고 운영하는 구조를 직접 구성하며 이해를 넓혀가고 있습니다.

한쪽만 파는 사람보다는, 도메인 지식과 구현을 함께 연결할 수 있는 사람이 되고 싶습니다.

대표 작업

지금까지 해 온 작업 중 홈페이지 첫 화면에 둘 만한 것들을 추렸습니다.

태양광 무선충전 자율주행 자동차 설계

학사 졸업논문 기반 작업입니다. 태양광 발전, 자기유도 무선충전, OpenCV 기반 차선 인식, CNN·SSD 객체 인식을 하나의 실험 시스템으로 묶었습니다.

Integrated Security Gateway

팀 BBQ 프로젝트입니다. 중앙 게이트웨이에서 인증·인가, Rate Limit, 로그 수집, ELK 분석, 경보와 추적을 묶어 보안 운영 흐름을 구성했습니다.

Coconut 클라우드 인프라 프로젝트

GCP, ArgoCD, Prometheus, Grafana, HPA, cronjob 기반 etcd 백업, Rclone 자동 백업을 묶어 GitOps형 인프라 협업 환경을 구축했습니다.

수소 혼소 화염 모니터링 AI

열화상 카메라, Raspberry Pi 5, YOLO v5를 이용해 수소 혼소 시 연소 상태와 기계적 위험을 예측하는 임베디드 모니터링 시스템을 다뤘습니다.

SAGD-CSS 프록시 모델링

석사논문 주제입니다. 불균질 오일샌드 저류층에서 복합 열공법의 시추공 배열과 생산운영을 설계하기 위해 DNN 기반 프록시 모델과 최적화를 적용했습니다.

메탄하이드레이트 재형성 모델링

열수 주입과 감압을 결합한 생산 과정에서 메탄하이드레이트의 재형성 가능성을 저류층 시뮬레이션으로 분석한 학회 포스터 작업입니다.

추후에는 각 카드에 상세 글, 발표자료, 회고를 연결할 예정입니다.

석사
